### 웹의 동작 개념
- 브라우저는 서버에 요청(Request) 을 보내고, 서버로부터 받은 HTML, CSS, JavaScript를 해석하여 화면을 그린다.
- 서버는 미리 만들어 둔 API를 통해 요청을 받아 처리한 뒤, HTML뿐만 아니라 JSON과 같은 데이터 형식으로도 응답할 수 있다.
- JavaScript가 API에서 받은 JSON 데이터를 이용해 화면 일부만 실시간으로 변경할 수 있다. (페이지 전체를 새로고침하지 않아도 데이터 교체 가능)

### HTML

HTML(HyperText Markup Language)은 태그(Tag) 를 사용하여 웹페이지의 구조를 만드는 언어이다.

사진, 텍스트, 링크 등 다양한 요소를 배치할 수 있다.

태그는 대부분 다음과 같이 시작 태그와 종료 태그로 이루어진다.

<p>안녕하세요.</p>

문서는 크게 head와 body로 구성된다.

#### head

주로 다음과 같은 정보를 작성한다.
- meta
- title
- link
- script

#### body
사용자에게 실제로 보이는 화면을 작성한다.

주로 사용하는 태그

- div
- p
- img
- a
- button


#### HTML의 트리(Tree) 구조

HTML에서는 어떤 요소 안에 어떤 요소가 들어있는지가 매우 중요하다.

```html
<div>
    <p>안녕하세요</p>
</div>
```

div는 부모(Parent)
p는 자식(Child)

HTML은 이러한 부모-자식 관계를 가지는 트리(Tree) 구조로 이루어진다.
