## 개념
- **Django**: 파이썬으로 작성된 오픈 소스 웹 프레임워크 
- 웹 개발의 복잡한 과정을 단순화하여 빠르고 안전하게 개발할 수 있도록 돕는다.
- **주요 철학**: "Don't Repeat Yourself (DRY)" 원칙을 따르며, 코드의 재사용성과 간결함을 중시
- **Batteries Included**: 데이터베이스 ORM, 인증 시스템, 관리자 패널 등 웹 개발에 필요한 필수 기능이 내장되어 있어 별도의 라이브러리 설치 없이 즉시 개발이 가능

## 구조 (MVT 패턴)
장고는 일반적인 MVC 패턴을 변형한 MVT(Model-View-Template) 패턴을 사용

- **Model**: 데이터 구조를 정의합니다. 데이터베이스의 테이블과 매핑되며, 데이터의 조작 및 제어를 담당(ORM 활용)
- **View**: 클라이언트의 요청을 받아 비즈니스 로직을 처리, 
 모델에서 데이터를 가져와 템플릿으로 전달하는 중간 관리자 역할을 합니다.
- **Template**: 사용자에게 보여질 화면(UI)을 담당. HTML 파일에 장고 템플릿 언어(DTL)를 사용하여 동적인 데이터를 렌더링.
- **추가 요소 (URLconf)**: 사용자의 요청(URL)을 적절한 뷰로 연결해주는 라우터 역할.
